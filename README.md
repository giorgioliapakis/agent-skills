# Agent Skills

Reusable agent skills for Claude Code and other AI coding agents.

## Install

```bash
npx skills add https://github.com/giorgioliapakis/agent-skills --skill keyword-researcher
```

## Skills

| Skill | Description |
|-------|-------------|
| [keyword-researcher](skills/keyword-researcher/) | DataForSEO CLI for keyword research, SERP analysis, and competitor intelligence |
